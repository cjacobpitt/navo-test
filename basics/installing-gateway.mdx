title: "Installing Gateway"
---

The Gateway application is available for use on the iPad and PC, with minor differences in appearance. Different devices require different methods of installation.

## Installing on the iPad

Gateway is available as an application on the App Store. The initial download of the application is free, as are subsequent updates.

<Steps>
  <Step>
    Sign into [www.dimins.com/](https://www.dimins.com/) using your user name and password.
  </Step>
  <Step>
    Click the <Tooltip tip="">DI-DOWNLOAD</Tooltip> link under the DI-Download section.

    ![Di Download Pn](/images/gateway-screenshots/di-download.png)
  </Step>
  <Step>
    In the list, find the DiveTab-PC file and click the version number.

    ![Di Download Gateway Pn](/images/gateway-screenshots/di-download-gateway.png)
  </Step>
  <Step>
    Your browser prompts you to run or save the file. Save the file, to a place of your choosing.

    A dialog box opens asking whether you want to save the file.

    <Note>
      Some systems choose a location automatically, typically the Downloads folder.
    </Note>
  </Step>
  <Step>
    Navigate to the folder where _Gateway-Setup.exe_ is stored, and double-click the file to start the setup.

    The Setup Wizard window opens.

    ![Gateway Install Welcome Pn](/images/gateway-screenshots/gateway-install-welcome.png)
  </Step>
  <Step>
    Follow the instructions in the setup wizard to complete the installation.
  </Step>
</Steps>
